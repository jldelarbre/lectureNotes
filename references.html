Les repos publics Maven
                Tour d’horizon :
                               https://www.deps.co/guides/public-maven-repositories/

                               Search site :
https://search.maven.org/
                Search site, all repos at a glance :
https://mvnrepository.com/
Dans la section « indexed repositories » :
Search on maven central :
https://mvnrepository.com/repos/central
browse maven central :
https://repo1.maven.org/maven2/
                                               Search on sonatype repo:
https://mvnrepository.com/repos/sonatype-releases
browse sonatype repo :
https://repo1.maven.org/maven2/
                               Exemple de recherche sur guava (utilisation des filtres dans la colonne de gauche) :
                                               Sans filtre :
                                                               https://mvnrepository.com/search?q=guava
                                               Seulement sur maven central :
                                                               https://mvnrepository.com/search?q=guava&repo=central
                                               Seulement les artifacts produits par google :
                                                               https://mvnrepository.com/search?q=guava&repo=central&d=com.google
